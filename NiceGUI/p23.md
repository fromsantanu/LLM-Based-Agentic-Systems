# **Lesson 23 ‚Äî Adding Charts and Graphs with NiceGUI + Plotly / Matplotlib**

## ‚≠ê What you will learn in this lesson

In this lesson, you will learn how to show **simple charts** inside a NiceGUI web page.
Charts help you **see data like sales, marks, or temperatures** in an easy way‚Äîjust like looking at a picture instead of reading numbers.

We will create:

* one chart using **Plotly** (very interactive and colorful)
* one chart using **Matplotlib** (simple and classic)

Both will appear on a NiceGUI web page.

# üß† Real-life example

Imagine you run a small shop. You wrote down how many items you sold each day:

* Monday ‚Üí 10
* Tuesday ‚Üí 20
* Wednesday ‚Üí 15

A picture (chart) of this data helps you quickly ‚Äúsee‚Äù which day was best.

## ‚úÖ Step 2: Create a NiceGUI page

We will make a page that shows:

1. A Plotly line chart
2. A Matplotlib bar chart

Here is the **full code** (kept simple):

```python
from nicegui import ui
import plotly.graph_objects as go
import matplotlib.pyplot as plt

# --- Plotly Chart ---
def make_plotly_chart():
    days = ["Mon", "Tue", "Wed"]
    sales = [10, 20, 15]

    fig = go.Figure(data=go.Scatter(x=days, y=sales, mode="lines+markers"))
    return fig

# --- Matplotlib Chart ---
def make_matplotlib_chart():
    days = ["Mon", "Tue", "Wed"]
    sales = [10, 20, 15]

    plt.figure(figsize=(4, 3))
    plt.bar(days, sales)
    return plt.gcf()

# --- NiceGUI Page ---
with ui.page("/"):
    ui.label("Sales Charts").classes("text-2xl")

    ui.label("Plotly Chart (Interactive)")
    ui.plotly(make_plotly_chart())

    ui.separator()
    ui.label("Matplotlib Chart (Simple)")
    ui.pyplot(make_matplotlib_chart())

ui.run()
```

# ‚ú® Explanation of the code (in very simple words)

### **1. Importing tools**

```python
from nicegui import ui
import plotly.graph_objects as go
import matplotlib.pyplot as plt
```

* `nicegui` ‚Üí helps us build web pages
* `plotly` ‚Üí helps us draw fancy charts
* `matplotlib` ‚Üí helps us draw simple bar/line charts

---

### **2. Building the Plotly chart**

```python
def make_plotly_chart():
    days = ["Mon", "Tue", "Wed"]
    sales = [10, 20, 15]

    fig = go.Figure(data=go.Scatter(x=days, y=sales, mode="lines+markers"))
    return fig
```

* We store days and sales in lists
* `Scatter` draws a line with points
* `fig` is the final chart we will show on the page

---

### **3. Building the Matplotlib chart**

```python
def make_matplotlib_chart():
    days = ["Mon", "Tue", "Wed"]
    sales = [10, 20, 15]

    plt.figure(figsize=(4, 3))
    plt.bar(days, sales)
    return plt.gcf()
```

* `plt.bar()` draws bars
* `gcf()` means ‚Äúget current figure‚Äù ‚Üí returns the picture

---

### **4. Creating the NiceGUI page**

```python
with ui.page("/"):
```
