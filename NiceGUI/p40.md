# **Step-by-Step**

---

## **1. Using Themes**

NiceGUI already has built-in light and dark themes.

We can switch the theme like this:

```python
from nicegui import ui

ui.colors(primary='blue', secondary='orange')
ui.dark_mode()

ui.label('This is dark mode UI')

ui.run()
```

### **Simple explanation**

* `ui.colors(...)` ‚Üí sets the main colors used in your app
* `ui.dark_mode()` ‚Üí turns everything into dark mode
* `ui.label` ‚Üí shows simple text
* `ui.run()` ‚Üí starts your app

---

## **2. Using Icons**

NiceGUI uses many ready-made icons (from Material Icons).
Just write the icon‚Äôs name.

```python
from nicegui import ui

with ui.row():
    ui.icon('home')
    ui.icon('settings')
    ui.icon('favorite')

ui.run()
```

### **Simple explanation**

* `ui.row()` ‚Üí keeps things in one horizontal line
* `ui.icon('home')` ‚Üí shows a home icon
* You can use many icons like `"search"`, `"person"`, `"info"`, `"chat"`, etc.

---

## **3. Adding Spacing (Padding and Margins)**

Spacing keeps your app neat and tidy.
Think of spacing as ‚Äúletting things breathe‚Äù.

```python
from nicegui import ui

with ui.column().classes('p-4 gap-4'):
    ui.label('Title')
    ui.button('Click Me')
    ui.label('Another text block')

ui.run()
```

### **Simple explanation**

* `.classes('p-4')` ‚Üí adds padding (space *inside* the box)
* `.classes('gap-4')` ‚Üí adds space *between* elements
* It makes things look more organized

---

## **4. Simple Animations**

Animations make elements appear smoothly.

```python
from nicegui import ui

with ui.column().classes('items-center gap-4'):
    ui.label('Hover over the box below üëá')

    ui.card().classes('p-4 hover:scale-105 transition-all duration-300'):
        ui.label('I grow when you hover!')

ui.run()
```

### **Simple explanation**

* `hover:scale-105` ‚Üí the box becomes slightly bigger when your mouse is over it
* `transition-all` ‚Üí the change happens smoothly
* `duration-300` ‚Üí animation lasts 0.3 seconds
* `ui.card()` ‚Üí a simple box with borders

This makes your UI feel more modern and interactive.

---

# **Complete Example (All 4 Features Together)**

```python
from nicegui import ui

ui.dark_mode()
ui.colors(primary="green")

with ui.column().classes('p-6 gap-4 items-center'):

    ui.label('Beautiful NiceGUI UI with Theme, Icons, Spacing, and Animation')
    
    ui.button('Click Me', icon='thumb_up') \
        .classes('p-3 hover:scale-110 transition-all duration-300')

    with ui.card().classes('p-4 gap-2 hover:shadow-2xl transition-all duration-300'):
        ui.icon('info')
        ui.label('This card has spacing, icon and animation')

ui.run()
```

### **What this example shows**

* dark theme
* custom color
* icons
* spacing using padding/gap
* small animations
* a card that glows when hovered

# **Tips & Best Practices**

### **1) Keep spacing consistent**

Use the same spacing (like `gap-4`) everywhere.
It gives a clean, professional look.

### **2) Do not overuse animations**

Small, soft animations look good.
Too many animations may look distracting.
