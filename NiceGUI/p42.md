# **Lesson 42: Adding Sidebar Navigation in NiceGUI**

### **What you will learn**

In this lesson, you will learn how to create a **sidebar menu** in your NiceGUI app.
A sidebar is the vertical menu you see on the left side of many apps ‚Äî like Gmail or YouTube.

It helps users move between different pages easily, without crowding the main screen.

Think of it like a **table of contents** in a book.
Instead of flipping pages randomly, you simply click the topic you want.

---

# **1. What is a Sidebar?**

A sidebar is a fixed vertical area that stays on the left side of your screen.
It usually contains:

* icons
* menu items
* links to different pages
* logout button or user profile

In NiceGUI, we use **`ui.drawer()`** to create a sidebar.

---

# **2. A Simple Sidebar Example**

### **Code**

```python
from nicegui import ui

with ui.drawer(side='left', bordered=True) as sidebar:
    ui.label('Menu').classes('text-xl font-bold')
    ui.link('Home', '/')
    ui.link('Patients', '/patients')
    ui.link('Settings', '/settings')

ui.label('Welcome to the main page!')

ui.run()
```

### **Explanation in simple words**

* `ui.drawer()` ‚Üí creates the sidebar
* `side='left'` ‚Üí shows it on left
* `bordered=True` ‚Üí gives a small border
* Inside the drawer we add:

  * `ui.label()` ‚Üí title
  * `ui.link()` ‚Üí clickable navigation links
* Outside the drawer we put the main content

---

# **3. Sidebar With Icons**

Icons make the menu easier to understand.

```python
from nicegui import ui

with ui.drawer(side='left', bordered=True):
    ui.label('Navigation').classes('text-xl font-bold mb-2')

    with ui.row().classes('items-center gap-2'):
        ui.icon('home')
        ui.link('Home', '/')

    with ui.row().classes('items-center gap-2'):
        ui.icon('group')
        ui.link('Patients', '/patients')

    with ui.row().classes('items-center gap-2'):
        ui.icon('settings')
        ui.link('Settings', '/settings')

ui.label("Main Content Area").classes('p-4')

ui.run()
```

### **Simple explanation**

* `ui.icon('home')` ‚Üí adds a small picture
* `ui.row()` ‚Üí keeps icon + text in one line
* Looks cleaner and easier to use

# **4. Responsive Sidebar (can be opened/closed)**

Sometimes you want a small button that **opens or closes** the sidebar.

### **Code**

```python
from nicegui import ui

with ui.drawer() as drawer:
    ui.label('Menu').classes('text-lg font-bold')
    ui.link('Dashboard', '/')
    ui.link('Reports', '/reports')

ui.button('‚ò∞ Menu', on_click=drawer.toggle).classes('m-4')

ui.label('Main area content...')

ui.run()
```

### **Simple explanation**

* `drawer.toggle` ‚Üí opens and closes the sidebar
* `‚ò∞` is a simple ‚Äúhamburger‚Äù menu icon
* Very useful on mobile screens

# **5. Full Sidebar Navigation Demo (Multiple Pages)**

### **Code**

```python
from nicegui import ui


# --- Sidebar Component ---
def sidebar_menu():
    with ui.drawer(side='left', bordered=True):
        ui.label('Clinic Menu').classes('text-xl font-bold mb-4')

        with ui.column().classes('gap-2'):
            ui.link('Home', '/')
            ui.link('Patients', '/patients')
            ui.link('Appointments', '/appointments')
            ui.link('Settings', '/settings')


# --- Pages ---
@ui.page('/')
def home_page():
    sidebar_menu()
    ui.label('üè† Home Page').classes('text-2xl p-4')


@ui.page('/patients')
def patients_page():
    sidebar_menu()
    ui.label('üë• Patients Page').classes('text-2xl p-4')


@ui.page('/appointments')
def appointments_page():
    sidebar_menu()
    ui.label('üìÖ Appointments Page').classes('text-2xl p-4')


@ui.page('/settings')
def settings_page():
    sidebar_menu()
    ui.label('‚öôÔ∏è Settings Page').classes('text-2xl p-4')


ui.run()
```

### **Simple explanation**

* We created **one reusable sidebar function** ‚Üí `sidebar_menu()`
* Each page calls this function
* This keeps the design uniform across your whole app
* You get multiple pages working with the same sidebar
