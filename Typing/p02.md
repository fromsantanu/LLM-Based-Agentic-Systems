## 2.2 `str`

## 2.3 `float`

Represents decimal numbers.

```python

    pi: float = 3.14159
    return pi * radius * radius

print(area_of_circle(3.5))  # âœ… 38.4844775
```

## 2.4 `bool`

Represents Boolean values (`True` or `False`).

```python

    return n % 2 == 0

print(is_even(4))   # âœ… True
print(is_even(7))   # âœ… False
```

## 2.5 `None`

Represents the absence of a value (like `null` in other languages).

```python
from typing import Optional

def find_user(user_id: int) -> Optional[str]:
    if user_id == 1:
        return "Admin"
    return None

print(find_user(1))   # âœ… "Admin"
print(find_user(2))   # âœ… None
```

> ðŸ”‘ Use `Optional[type]` when a function might return a value **or** `None`.

## 2.6 `Any`

Represents **any type**.
Use it when you donâ€™t know (or donâ€™t care) about the type.

```python

def print_value(value: Any) -> None:
    print(f"The value is {value}")

print_value(42)         # int
print_value("hello")    # str
print_value([1, 2, 3])  # list
```

âš ï¸ **Warning**: Using `Any` skips type checking, so prefer specific types when possible.

## 2.7 Example: Mixing Types

```python
from typing import Any, Optional

def process_input(x: Any) -> Optional[str]:
    if isinstance(x, int):
        return f"Got an integer: {x}"
    elif isinstance(x, str):
        return f"Got a string: {x}"
    else:
        return None

print(process_input(100))       # "Got an integer: 100"
print(process_input("Python"))  # "Got a string: Python"
print(process_input(3.14))      # None
```
